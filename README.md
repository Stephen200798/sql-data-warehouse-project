# Modern Data Warehouse for Sales Analytics

A project to design and implement a modern data warehouse using SQL Server, aimed at consolidating sales data for business intelligence, performance tracking, and strategic decision-making.

---

## 📌 Project Objectives

- Centralize sales data from various sources (ERP, CRM, flat files)
- Design a scalable star-schema data warehouse
- Implement robust ETL processes using SSIS
- Enable self-service and dashboard-based analytics

---

## ✅ Project Requirements

- SQL Server instance 
- SQL Server Integration Services (SSIS)
- Basic ETL and T-SQL knowledge
- Power BI for visualization
- Git for version control

---

## ⚙️ Specifications

- **ETL Layer**: Handles extraction, cleansing, transformation, and loading
- **Schema**: Star schema with fact and dimension tables
- **Staging Area**: Temporary data holding before transformations
- **Data Sources**: CRM, ERP systems, Excel/CSV files
- **Warehouse**: Optimized for querying and analytics

---

## 📊 Analytics & Reporting

The warehouse supports the following business insights:

- 📈 Total Sales Over Time  
- 🛒 Top-Selling Products  
- 🌍 Sales by Region  
- 👥 Customer Purchase Trends  
- 📅 Month-over-Month & Year-over-Year Comparisons  

Reports are built in Power BI and SSRS with filtering and drill-downs enabled.
